SIMON GAME

Level: 1

Instructions


  1. The game will show a sequence of colors.
  2. You need to repeat the sequence by clicking the matching buttons.
  3. If you get the sequence right, the game progresses to the next round.
  4. Strict mode: if you make a mistake, the game will restart from the beginning. In normal mode, you can retry without losing your progress..
